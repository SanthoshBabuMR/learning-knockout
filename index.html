notes on Knockout

  • Unless and until, observables are defined in js and "ko.applyBindings" is invoked, two-way binding doesn't happen
  • Two-way binding occurs only for input, textarea elements from DOM. A div element with "contenteditable" attribute set to true will not auto update
  • Inbuild ko template's mechanism require that template name be enclosed in quotes, when referenced within script tags using type as "text/html"
